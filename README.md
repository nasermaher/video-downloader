# برنامج تحميل الفيديوهات (Video Downloader)

برنامج سطح مكتب بواجهة رسومية (GUI) مبني بلغة Python لتحميل الفيديوهات وقوائم التشغيل من يوتيوب وأي موقع آخر يدعمه [yt-dlp](https://github.com/yt-dlp/yt-dlp) (فيميو، تويتر/X، وغيرها). يعرض الجودات المتاحة فعليًا قبل التحميل، ويتيح اختيار مكان الحفظ على الجهاز.

> A desktop GUI app to download videos and playlists from YouTube and any site supported by yt-dlp (Vimeo, Twitter/X, and more), with real available-quality selection before downloading.

## المزايا

- تحميل فيديو مفرد أو قائمة تشغيل كاملة، مع إمكانية تحديد فيديوهات معينة من القائمة (تحديد الكل / إلغاء تحديد الكل).
- عرض الجودات المتاحة فعليًا لكل فيديو، موضّح جنب كل خيار هل معاه صوت أو لا (فيديو فقط / صوت فقط / فيديو + صوت)، مع دمج تلقائي لأفضل صوت متاح عند اختيار جودة فيديو بدون صوت.
- اختيار مسار الحفظ عبر نافذة تصفح الملفات، مع حفظ آخر مسار مستخدم.
- شريط تقدم لكل تحميل (نسبة، سرعة).
- **إيقاف مؤقت / استكمال**: يوقف التحميل الجاري ويحتفظ بالجزء المحمّل، ويكمل من نفس النقطة لاحقًا (بدل البدء من الصفر).
- **استكمال تلقائي للتحميلات المنقطعة** حتى بعد إغلاق البرنامج وإعادة فتحه.
- **تحميل الترجمة** تلقائيًا كملف SRT بنفس اسم ملف الفيديو.
- **تحميل وصف الفيديو كامل** (الوصف والروابط الموجودة فيه) كملف TXT بنفس اسم ملف الفيديو.
- **قائمة انتظار (Queue)** لتحميل عدة روابط منفصلة دفعة واحدة، بالتتابع أو بالتوازي (بحد أقصى متزامن قابل للتحديد)، مع دعم إيقاف مؤقت/استكمال لها أيضًا.
- فاصل زمني تلقائي بين كل تحميل والتالي لتخفيف الضغط على السيرفرات.
- **وضع داكن / فاتح**، و**دعم لغتين** (عربي وإنجليزي) قابلتين للتبديل من شريط القوائم.
- يعمل مع أي موقع فيديو يدعمه yt-dlp، مش يوتيوب بس.

## المتطلبات

- Python 3.10 أو أحدث (تم اختباره أيضًا على 3.14).
- [ffmpeg](https://www.gyan.dev/ffmpeg/builds/) مثبت على الجهاز ومتاح في PATH (مطلوب لدمج جودات الفيديو والصوت، وتحويل الترجمة لصيغة SRT).
- المكتبات في `requirements.txt` (PyQt5، yt-dlp).

## التشغيل من الكود المصدري

```bash
# 1. إنشاء بيئة افتراضية (اختياري لكن موصى به)
python -m venv .venv
.venv\Scripts\activate        # ويندوز
# source .venv/bin/activate   # ماك/لينكس

# 2. تثبيت المتطلبات
pip install -r requirements.txt

# 3. تشغيل البرنامج
python main.py
```

## طريقة الاستخدام

1. الصق رابط فيديو مفرد أو رابط قائمة تشغيل في الحقل العلوي واضغط **جلب المعلومات**.
2. لو الرابط قائمة تشغيل، هتظهر كل الفيديوهات مع صناديق اختيار — استخدم **تحديد الكل** / **إلغاء تحديد الكل** أو حدد فيديوهات معينة يدويًا.
3. اختر **الجودة / الصيغة** المطلوبة من القائمة المنسدلة (موضّح جنب كل خيار هل فيه صوت أو لا).
4. فعّل **تحميل الترجمة** و/أو **تحميل وصف الفيديو** لو محتاجهم، مع تحديد رموز لغات الترجمة (مثال: `ar,en`).
5. اختر مسار الحفظ عبر **استعراض...**.
6. اضغط **بدء التحميل**. تقدر توقف مؤقتًا بـ **إيقاف مؤقت** وتكمل لاحقًا بنفس الزرار، أو تلغي التحميل بالكامل بـ **إيقاف**.

### تبويب قائمة الانتظار

لتحميل عدة روابط منفصلة دفعة واحدة: افتح تبويب **قائمة الانتظار**، الصق الروابط (رابط واحد في كل سطر) واضغط **إضافة للقائمة**، اختر **متتابع** أو **متوازي** (مع تحديد أقصى عدد متزامن)، ثم اضغط **بدء تحميل القائمة**.

### اللغة والمظهر

من شريط القوائم أعلى النافذة: **اللغة** للتبديل بين العربي والإنجليزي فوريًا، و**المظهر** للتبديل بين الفاتح والداكن.

## بنية المشروع

```
main.py                  نقطة تشغيل البرنامج
gui/
  main_window.py         النافذة الرئيسية (تبويب التحميل)
  queue_widget.py         عنصر واجهة قائمة الانتظار
core/
  extractor.py            استخلاص معلومات الفيديو/القائمة والصيغ المتاحة عبر yt-dlp
  downloader.py           تنفيذ التحميل في خيط منفصل (تقدم، إيقاف مؤقت/استكمال، ترجمة، وصف)
  queue_manager.py        تنسيق تحميل عدة روابط بالتتابع أو بالتوازي
  playlist.py             أدوات مساعدة لقوائم التشغيل
  settings.py             حفظ واسترجاع إعدادات المستخدم (JSON)
  paths.py                تحديد مسارات بيانات التطبيق وملف ffmpeg المرفق
  i18n.py                 طبقة الترجمة (عربي/إنجليزي)
  theme.py                أنماط المظهر الفاتح/الداكن
  logger.py                تسجيل الأحداث والأخطاء
tests/                    اختبارات pytest (تعمل بدون اتصال إنترنت حقيقي عبر تزييف yt-dlp)
main.spec                 ملف تحزيم PyInstaller
```

## تشغيل الاختبارات

```bash
pip install -r requirements-dev.txt
pytest -q
```

## تحزيم البرنامج كملف تنفيذي (exe)

```bash
pip install -r requirements-dev.txt
pyinstaller main.spec
```

الناتج هيكون في `dist/VideoDownloader.exe`. لو عايز البرنامج يشتغل بدون ما يحتاج المستخدم يثبت ffmpeg بنفسه، حمّل نسخة [ffmpeg](https://www.gyan.dev/ffmpeg/builds/) الثابتة (static build) وحط `ffmpeg.exe` في نفس مجلد الـ exe الناتج — البرنامج بيدور عليه هناك تلقائيًا.

الإعدادات والسجلات بتتخزن في `%APPDATA%\VideoDownloader` بعد التحزيم.

## ملاحظة قانونية

استخدم البرنامج فقط لتحميل محتوى مسموح لك تحميله (محتواك الشخصي، أو محتوى ذو ترخيص مفتوح، أو بإذن صاحب الحقوق). تحميل محتوى محمي بحقوق الملكية دون إذن قد يخالف شروط استخدام المنصة والقوانين المحلية.
